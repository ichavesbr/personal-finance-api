# Personal Finance API
> RESTful API for personal finance management built with Node.js, TypeScript and Express.

## Overview
API for managing personal finances — track income, expenses and monthly summaries organized by category.

> **Status:** MVP in development. The project is being built incrementally — features like Docker, tests, validation and API docs will be added over time.

![Last Commit](https://img.shields.io/github/last-commit/ichavesbr/personal-finance-api)

## Current Tech Stack
- **Runtime:** Node.js 20+
- **Language:** TypeScript
- **Framework:** Express
- **Database:** PostgreSQL
- **Auth:** JWT
- **Validation:** Zod

## Planned Additions
- **ORM:** Prisma
- **Testing:** Jest + Supertest
- **Docs:** Swagger/OpenAPI
- **Containerization:** Docker

## Getting Started

### Prerequisites
- Node.js 20+
- PostgreSQL running locally

### Installation
```bash
git clone https://github.com/ichavesbr/personal-finance-api.git
cd personal-finance-api
cp .env.example .env
npm install
npm run dev
```

### Environment Variables
See `.env.example`
